# Zenn連携用リポジトリ設定

このリポジトリはZennのGitHub連携機能を使用しています。

## ディレクトリ構造

```
jpycwallet-x402/
├── articles/          # 記事を格納するディレクトリ
│   ├── *.md          # 各記事ファイル
│   └── README.md     # 記事管理用README
├── books/            # 本を格納するディレクトリ（現在は未使用）
│   └── .gitkeep
└── README.md         # プロジェクトのREADME
```

## Zenn連携の確認

このリポジトリは以下の要件を満たしています：

- ✅ `articles/` ディレクトリが存在
- ✅ `books/` ディレクトリが存在
- ✅ 記事ファイルが適切なフォーマット（Front Matter付き）

## トラブルシューティング

### 「booksディレクトリが見つかりませんでした」エラー

→ 解決済み：`books/` ディレクトリを作成しました

### リポジトリ連携が反映されない場合

1. Zennの設定ページでリポジトリ連携を確認
2. デフォルトブランチが `main` になっているか確認
3. `articles/` と `books/` が**ルートディレクトリ直下**にあるか確認
